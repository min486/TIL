# CLI

명령어 인터페이스

## 기본 명령어

현재 디렉토리 출력

```
git init
```

디렉토리 이동

```
cd 디렉토리이름
```

현재 디렉토리

```
.
```

상위 디렉토리

```
..
```

목록

```
ls (list)
```

디렉토리 생성

```
mkdir (make directory)
```

파일 생성

```
touch 
```

파일 삭제하기

```
rm 파일명
```

폴더 삭제하기

```
rm –r 폴더명
```

## git 명령어

저장소 처음 만들때

```
git init
```

버전을 기록할 때

```
git add
```

- 현재 디렉토리 전체 add

  ```
  git add .

- 해당 파일 add

  ```
  git add 파일명

상태 확인할 때

- 1통, 2통 확인

  ```
  git status
  ```

- 커밋 확인

  ```
  git log
